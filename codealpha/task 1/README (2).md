## ✨ About

**GalleryHub** is a clean, modern visual collection platform where you can browse a curated gallery of stunning photos, filter them by category, upload your own images, and download favourites — all within a beautiful, responsive interface.

> Built as part of the **CodeAlpha Frontend Development Internship** program.

## 📸 Features

| Feature | Description |
|--------|-------------|
| 🗂️ **Category Filter** | Browse photos by Nature, Architecture, Travel, Portrait, Abstract & Food |
| 📤 **Upload Photos** | Add your own images with name and category |
| 🔍 **Lightbox Viewer** | Click any photo for a full-screen preview |
| ⬇️ **Download** | Save any image directly to your device |
| 📱 **Responsive Design** | Works seamlessly on mobile, tablet & desktop |
| 🎨 **Clean UI** | Minimal, modern and distraction-free layout |

---

## 🗂️ Gallery Categories

- 🌿 **Nature** — Aurora, Mountains, Lavender Fields
- 🏛️ **Architecture** — Eiffel Tower, Colosseum, City Skyline
- ✈️ **Travel** — Santorini, Tokyo, Bali
- 🧑 **Portrait** — Golden Hour, Studio Monochrome
- 🎨 **Abstract** — Liquid Colors, Paint Splash, Smoke Art
- 🍽️ **Food** — Gourmet Burger, Sushi, Pasta, Dessert

---

## 🛠️ Tech Stack

- **HTML5** — Structure & semantics
- **CSS3** — Styling, grid layout, animations
- **Vanilla JavaScript** — Dynamic filtering, upload, lightbox & download logic

## ⚙️ Getting Started

```bash

# 2. Open in browser
cd CodeAlpha_Gallery_Hub
open index.html
```

> No dependencies. No build tools. Just open and run! 🎉
