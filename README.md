## 🖼️ Task 1 — GalleryHub

<div align="center">

[![Live Demo](https://img.shields.io/badge/🌐%20Live%20Demo-Visit%20Now-6C63FF?style=for-the-badge)](https://yashsoni972.github.io/CodeAlpha_Gallery_Hub/)
[![GitHub Repo](https://img.shields.io/badge/📂%20Source%20Code-GitHub-181717?style=for-the-badge&logo=github)](https://github.com/yashsoni972/CodeAlpha_Gallery_Hub)

</div>

**GalleryHub** is a fully interactive visual collection platform where users can browse, upload, filter, and manage images — all beautifully organized into categories with a modern, responsive UI.

### ✨ Features

- 📸 **18 curated photos** across 6 categories — Nature, Architecture, Travel, Portrait, Abstract & Food
- 🔍 **Category filter** — instantly filter images by type with one click
- ⬆️ **Photo upload** — drag & drop or choose a file to add your own photos
- 🔎 **Lightbox viewer** — click any photo to view full-screen with a download option
- 📱 **Fully responsive** — works seamlessly on desktop, tablet & mobile
- 🎨 Clean, modern card-based UI with smooth hover animations

## 🧮 Task 2 — Calculator Pro

<div align="center">

[![Live Demo](https://img.shields.io/badge/🌐%20Live%20Demo-Visit%20Now-FF6584?style=for-the-badge)](https://yashsoni972.github.io/CodeAlpha_Calculator-Pro/)
[![GitHub Repo](https://img.shields.io/badge/📂%20Source%20Code-GitHub-181717?style=for-the-badge&logo=github)](https://github.com/yashsoni972/CodeAlpha_Calculator-Pro)

</div>

**Calculator Pro** is a sleek, feature-rich scientific calculator with a clean modern UI and powerful computation capabilities for both standard and advanced mathematical operations.

### ✨ Features

- ➕ **Basic arithmetic** — Addition, Subtraction, Multiplication, Division
- 🔬 **Scientific functions** — sin, cos, tan, log, ln, √, x², xʸ
- 📋 **Calculation history** — view all past calculations with a clear option
- ⚡ **Math constants** — built-in π (Pi) and e (Euler's number)
- ⌫ **Backspace & AC** — quick correction controls for smooth input
- 📱 **Responsive layout** — works perfectly on all screen sizes
